Black-Scholes-Merton Formula

European Call Option Pricing Formula

\[ c = S_{0}N\left( d_{1} \right) - Xe^{- \text{rT}}N\left( d_{2} \right) \]

е…¶дёӯпјҡ

\[ d_{1} = \frac{\ln\left( \frac{S_{0}}{X} \right) + \left( r + \frac{\sigma^{2}}{2} \right)T}{\sigma\sqrt{T}} \]

\[ d_{2} = \frac{\ln\left( \frac{S_{0}}{X} \right) + \left( r - \frac{\sigma^{2}}{2} \right)T}{\sigma\sqrt{T}} = d_{1} - \sigma\sqrt{T} \]

дё”\(N\left( x \right)\) иЎЁзӨәж ҮеҮҶжӯЈжҖҒеҲҶеёғ\(N\left( 0,1 \right)\)зҡ„жҰӮзҺҮеҖј

жҺЁеҜјBlack-Scholes-Merton Formulaпјҡ

Black-Scholes-Merton Formula еҒҮи®ҫ

  • иӮЎзҘЁд»·ж јжңҚд»ҺеҮ дҪ•еёғжң—иҝҗеҠЁ / еҜ№ж•°жӯЈжҖҒеҲҶеёғпјҢеҚі \[ dS=\mu Sdt+\sigma S dW_t \Rightarrow \frac{dS}{S}=\mu dt+\sigma dW_t \] е…¶дёӯ \(\{W_t \}_{t \geq 0}\) жңҚд»Һж ҮеҮҶеёғжң—иҝҗеҠЁзҡ„йҡҸжңәиҝҮзЁӢгҖӮ \[ dW_t=\epsilon \sqrt{dt} \] еҲ©з”ЁеүҚиҝ° \(It \hat{o}\) еј•зҗҶпјҢеҜ№дәҺ \(f(t,S)\) пјҢе…¶дёӯ S жңҚд»ҺдёҠйқўзҡ„иҝҮзЁӢпјҢеҲҷ \[ df=(\frac{\partial f}{\partial S}\mu S+\frac{\partial f}{\partial t}+\frac{1}{2} \frac{\partial^2 f}{\partial S^2} \sigma^2 S^2)dt+\frac{\partial f}{\partial S} \sigma S dW_t \]

    еҪ“ \(f(t,S)=\ln S\) ж—¶пјҢд»Је…Ҙеҫ—пјҡ \[ d\ln S=(\mu -\frac{\sigma^2}{2}) dt+\sigma dW_t \] еҚі \(f(t,S)=\ln S\) ж»Ўи¶іGeneralized Wiener Process / еёҰжјӮз§»зҡ„ Brown иҝҗеҠЁпјҢжјӮз§»зҺҮдёәеёёж•° \(\mu -\frac{\sigma^2}{2}\) пјҢжіўеҠЁзҺҮдёәеёёж•° \(\sigma^2\) гҖӮ

    еҚі \(\ln S\) еңЁ \([0,T]\) ж—¶й—ҙеҸҳеҢ–жңҚд»ҺжӯЈжҖҒеҲҶеёғпјҡ \[ \ln (\frac{S_T}{S_0}) \sim \Phi((\mu -\frac{\sigma^2}{2})T,\sigma^2T) \Rightarrow S_T=S_0e^{(\mu -\frac{\sigma^2}{2})T+\sigma W_T} \] жҳҫз„¶жңү \[ E(S_T)=S_0e^{\mu T} \]

иӮЎзҘЁд»·ж јжңҚд»ҺеҮ дҪ•еёғжң—иҝҗеҠЁпјҢеҲҷиӮЎзҘЁзҡ„иҝһз»ӯеӨҚеҲ©ж”¶зӣҠзҺҮ x жңҚд»ҺжӯЈжҖҒеҲҶеёғгҖӮ \[ S_t=S_0e^{xT} \Rightarrow x=\frac{1}{T} \ln (\frac{S_T}{S_0}) \Rightarrow x \sim \Phi (\mu -\frac{\sigma^2}{2},\frac{\sigma^2}{T}) \]

иҝҷйҮҢйңҖиҰҒеҢәеҲҶпјҡ

иӮЎзҘЁзҡ„жңҹжңӣж”¶зӣҠзҺҮ / йў„жңҹж”¶зӣҠзҺҮ \(\mu\) е№¶дёҚзӯүдәҺд»Ҙиҝһз»ӯеӨҚеҲ©и®Ўз®—зҡ„ж”¶зӣҠзҺҮзҡ„жңҹжңӣеҖј \(\mu-\frac{\sigma^2}{2}\) пјҢе®һйҷ…дёҠеӣ дёә

\[E[\ln(S_T)] = \ln S_0+\mu - \frac{\sigma^2}{2} < \ln E[S_T] = \ln S_0+\mu \]

  • еҸҜд»ҘеҚ–з©әиҜҒеҲёпјҢе№¶дё”еҸҜд»Ҙе®Ңе…ЁдҪҝз”ЁжүҖеҫ—ж”¶е…Ҙ

  • ж— дәӨжҳ“иҙ№з”ЁгҖҒж— зЁҺж”¶пјҢжүҖжңүиҜҒеҲёеҸҜд»Ҙж— йҷҗеҲҶеүІ

  • жңҹжқғжңҹйҷҗеҶ…иӮЎзҘЁдёҚж”Ҝд»ҳиӮЎжҒҜ

    иҝҷдёҖжқЎеҗҺз»ӯеҸҜд»Ҙж”ҫжқҫдёәвҖңиӮЎзҘЁж”Ҝд»ҳиӮЎжҒҜзҡ„жңҹжқғе®ҡд»·жЁЎеһӢвҖқ

  • дёҚеӯҳеңЁж— йЈҺйҷ©еҘ—еҲ©жңәдјҡ

  • иҜҒеҲёдәӨжҳ“иҝһз»ӯиҝӣиЎҢ

  • зҹӯжңҹж— йЈҺйҷ©еҲ©зҺҮ r дёәеёёж•°пјҢе№¶еҜ№жүҖжңүжңҹйҷҗеқҮзӣёеҗҢгҖӮ

    иҝҷдёҖжқЎеҗҺз»ӯеҸҜд»Ҙж”ҫжқҫдёәвҖңr дёәе…ідәҺж—¶й—ҙ t зҡ„еҮҪж•°вҖқжҲ–вҖңr дёәжҹҗдёӘйҡҸжңәиҝҮзЁӢвҖқпјҢиҝҷйғЁеҲҶиҜҰи§ҒеҗҺз»ӯзҡ„еҲ©зҺҮжЁЎеһӢ

еҒҸеҫ®еҲҶж–№зЁӢеҪўејҸзҡ„Black-Scholes-Merton Equation

\[ \frac{\partial f}{\partial t}+rS\frac{\partial f}{\partial S} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 f}{\partial S^2}=rf \]

е…¶дёӯ \(f(t,S)\) дёәе…ідәҺ t е’Ң S зҡ„еҮҪж•°пјҢиЎЁзӨәдҫқиө–дәҺ S зҡ„иЎҚз”ҹдә§е“Ғд»·ж јгҖӮ

еҜ№дәҺжҹҗдёҖзү№е®ҡзҡ„йҮ‘иһҚиЎҚз”ҹдә§е“ҒпјҢдёҠиҝ°ж–№зЁӢзҡ„и§ЈдёҺе…¶иҫ№з•ҢжқЎд»¶жңүе…іпјҢиҫ№з•ҢжқЎд»¶е®ҡд№үдәҶйҮ‘иһҚиЎҚз”ҹе“ҒеңЁ S е’Ң t еңЁиҫ№з•ҢдёҠзҡ„еҸ–еҖјгҖӮ

еҜ№дәҺ European Call Option иҫ№з•ҢжқЎд»¶пјҡ \[ f=\max(S_T-K,0) \quad еҪ“ t=T ж—¶ \] е®№жҳ“еҫ—еҲ°е…¶жңҹжқғд»·ж јзҡ„и§ЈжһҗејҸгҖӮеҗҢзҗҶеҜ№ European Put Option гҖӮ

иҜҒжҳҺпјҡ

дёәж¶ҲйҷӨ Wiener Process зҡ„ \(dW_t\) йЎ№пјҢжһ„йҖ иӮЎзҘЁе’Ңжңҹжқғзҡ„жҠ•иө„з»„еҗҲпјҢе…¶д»·еҖјдёә \(\Pi=-f+\frac{\partial f}{\partial S}\)

  • \(-1\) иЎҚз”ҹдә§е“Ғ
  • \(+\frac{\partial f}{\partial S}\) иӮЎзҘЁ

еҲҷ \(d\Pi=-df+\frac{\partial f}{\partial S} dS \overset{\text{д»Је…ҘдёҠиҝ°}It\hat{o} \text{еј•зҗҶ}}{\Rightarrow} d\Pi=(-\frac{\partial f}{\partial t}-\frac{1}{2} \frac{\partial^2 f}{\partial S^2} \sigma^2 S^2 ) dt\) гҖӮ

жҠ•иө„з»„еҗҲеңЁжһҒзҹӯзҡ„ж—¶й—ҙеҶ…жҳҜж— йЈҺйҷ©зҡ„пјҢеӣ дёәдёҚеҗ«йҡҸжңәйЎ№гҖӮгҖҗд»…еңЁж— з©·е°Ҹзҡ„еҢәй—ҙеҶ…дёәж— йЈҺйҷ©пјҢдёәдҝқиҜҒиҜҒеҲёз»„еҗҲж— йЈҺйҷ©пјҢйңҖиҰҒз»ҸеёёеҜ№жҠ•иө„з»„еҗҲдёӯзҡ„иЎҚз”ҹдә§е“Ғе’ҢиӮЎзҘЁзҡ„жҜ”дҫӢеҒҡеҮәи°ғж•ҙпјҢеҚі \(\frac{\partial f}{\partial S}\) е…¶е®һжҳҜеҸ‘з”ҹеҸҳеҢ–зҡ„гҖ‘

з”ұеҒҮи®ҫпјҢиҜҒеҲёжҠ•иө„з»„еҗҲеҝ…йЎ»иҺ·еҸ–дёҺе…¶д»–зҹӯжңҹж— йЈҺйҷ©еҖәеҲёзӣёеҗҢзҡ„зһ¬ж—¶ж”¶зӣҠзҺҮпјҢеҚі\(d\Pi=r\Pi dt\)

е…¶дёӯ r дёәж— йЈҺйҷ©еҲ©зҺҮпјҢеёҰе…Ҙж•ҙзҗҶеҚіеҸҜеҫ—дёҠиҝ°еҒҸеҫ®еҲҶж–№зЁӢгҖӮ

йЈҺйҷ©дёӯжҖ§жқЎд»¶дёӢзҡ„Black-Scholes-Merton Formula

дёәдҪ•и®ҫе®ҡеңЁйЈҺйҷ©дёӯжҖ§дёӢ
жҺЁеҜјEuropean Option

жҺЁеҜјпјҡ

еҲ©з”ЁйЈҺйҷ©дёӯжҖ§е®ҡд»·ж–№жі•пјҢеңЁж— йЈҺйҷ©жөӢеәҰ Q дёӢпјҢжңҹжқғеҲ°жңҹзҡ„жңҹжңӣеҖјдёә \(E_{Q}[\max(S_T - X,0)]\) пјҢж•… European Call Option д»·ж јдёәд»Ҙж— йЈҺйҷ©еҲ©зҺҮиҙҙзҺ°еҗҺзҡ„зҺ°еҖј \[ c = e^{- \text{rT}}E_{Q}[\max(S_T - X,0)] \]

жҢүз…§ John Hull зҡ„жҺЁеҜјж–№жі•пјҢеҪ“ \(S_T\) жңҚд»ҺеҜ№ж•°жӯЈжҖҒеҲҶеёғж—¶пјҢи®ҫ \(\ln S_T\) зҡ„ж–№е·®дёә \(\omega\) пјҢеқҮеҖјдёә \(m=\ln E[S_T]-\frac{\omega^2}{2}\) пјҢеҲҷ \[ E[\max(S_T - X,0)]=E[S_T]N(d_1)-XN(d_2) \] е…¶дёӯ \[ d_{1} = d_{2} + \omega = \frac{1}{\omega}( \ln(\frac{E[S_T]}{X}) + \frac{\omega^2}{2}) \] иҝҷдёҖе…¬ејҸжҺЁеҜјеҰӮдёӢпјҡ \[ E[\max(S_T - X,0)] = \int_X^{+\infty} (S_T - X) g(S_T) dS_T \overset{Q=\frac{\ln S_T-m}{\omega} \sim N(0,1)}{=} \int_{\frac{\ln X -m}{\omega}}^{+\infty} e^{Q\omega+m} h(Q) dQ -K\int_{\frac{\ln X -m}{\omega}}^{+\infty} h(Q) dQ \] е…¶дёӯ \(h(Q)\) дёәж ҮеҮҶжӯЈжҖҒеҲҶеёғзҡ„жҰӮзҺҮеҜҶеәҰеҮҪж•°гҖӮеұ•ејҖжҰӮзҺҮеҜҶеәҰеҮҪж•°пјҢз”ұдәҺ \[ e^{Q\omega+m} h(Q)=e^{m+\frac{\omega^2}{2}}h(Q-\omega) \] д»Је…Ҙи®Ўз®—ж•ҙзҗҶеҸҜеҫ—дёҠиҝ°е…¬ејҸгҖӮ

е®һйҷ…дёҠпјҢд»Је…Ҙ \(E[S_T]=S_0e^{rT}\) е’Ң \(\omega=\sigma \sqrt{T}\) еҚіеҸҜеҫ—еҲ°з»“жһңгҖӮ

B-S Formula жҺЁе№ҝпјҡ

European Call Optionпјҡ

\[ c = S_{0}N\left( d_{1} \right) - Xe^{- rT}N\left( d_{2} \right) \]

е…¶дёӯ \(N(d_2)\) иЎЁзӨәйЈҺйҷ©дёӯжҖ§жқЎд»¶дёӢжңҹжқғиў«иЎҢдҪҝзҡ„жҰӮзҺҮпјҢжҠ•иө„дәҺиӮЎзҘЁзҡ„иө„йҮ‘жҖ»йўқдёә \(S_{0}N\left( d_{1} \right)\) пјҢе…¶дёӯ \(N(d_1)\) дёәиҙӯд№°иӮЎзҘЁзҡ„ж•°йҮҸпјҢжҠ•иө„дәҺж— йЈҺйҷ©иө„дә§зҡ„иө„йҮ‘жҖ»йўқдёәпјҲеҖҹе…Ҙпјү \(- Xe^{- rT}N\left( d_{2} \right)\) гҖӮ

European Put Optionпјҡ

ж №жҚ®ж¬§ејҸжңҹжқғе№ід»·е…ізі»пјҡ \[ c + Xe^{- rT} = p + S \]

ж•…жҲ‘д»¬жҺЁеҮә

\[ p = Xe^{- \text{rT}}N\left( - d_{2} \right) - S_{0}N\left( - d_{1} \right) \]

е…¶дёӯ \(d_{1},d_{2}\) дёҺдёҠиҝ°е…¬ејҸж„Ҹд№үзӣёеҗҢ

жіўеҠЁзҺҮ \(\sigma\) и®Ўз®—

\[ \ln S_{t}\left( w \right) - \ln S_{0}\sim N\lbrack(r - \frac{\sigma^{2}}{2})t,\sigma^{2}t\rbrack \]

  • еҺҶеҸІжіўеҠЁзҺҮпјҡйҰ–е…Ҳи®Ўз®—еҜ№ж•°ж”¶зӣҠзҺҮпјҢе…¶ж¬Ўи®Ўз®—ж ·жң¬ж ҮеҮҶе·®пјҢжңҖеҗҺжҢүе№ҙиҝӣиЎҢж ҮеҮҶеҢ–гҖӮ

  • йҡҗеҗ«жіўеҠЁзҺҮпјҡеҲ©з”ЁеёӮеңәжңҹжқғзҡ„жҠҘд»·еҸҠBвҖ”Sе…¬ејҸпјҢжҸ’еҖјд»Је…Ҙи®Ўз®—гҖӮ

  • жҢҮж•°еҠ жқғжіўеҠЁзҺҮпјҡеҲ©з”ЁжңҖиҝ‘ж—¶еҲ»зҡ„жіўеҠЁзҺҮеҸҠд»ҘеүҚдёҚеҗҢж—¶еҲ»жіўеҠЁзҺҮзҡ„еҠ жқғе’ҢгҖӮ

з•Ңе®ҡдәҢеҸүж ‘жЁЎеһӢдёӯзҡ„еҸӮж•°

\[ u = e^{\sigma\sqrt{\frac{T}{N}}\ } \text{with probability } p = \frac{1}{2}\left( 1 + \frac{\mu}{\sigma}\sqrt{\frac{T}{N}} \right) \]

\[ d = \frac{1}{u}\text{with probability } p = \frac{1}{2}\left( 1 - \frac{\mu}{\sigma}\sqrt{\frac{T}{N}} \right) \]

\[ \text{while }\Delta t = \frac{T}{N} \]

иҖҢ Hull-Whiteз®—жі•йҮҮеҸ–дәҶдёҚеҗҢзҡ„дәҢеҸүж ‘жЁЎеһӢеҸӮж•°гҖӮ

жңүзҰ»ж•ЈзәўеҲ©зҡ„ж¬§ејҸжңҹжқғзҡ„B-Sе…¬ејҸ

з”ЁиӮЎзҘЁзҡ„еёӮеңәд»·ж јеҮҸеҺ»иӮЎзҘЁеңЁжңҹжқғеҲ°жңҹж—Ҙд№ӢеүҚеҲҶй…Қзҡ„зәўеҲ©зҡ„зҺ°еҖјдҪңдёәиӮЎд»·д»Је…ҘеҲ°B-Sе…¬ејҸдёӯпјҢд»ҺиҖҢеҫ—еҲ°ж¬§ејҸжңҹжқғзҡ„д»·еҖјгҖӮ

еҚіпјҡ \[ c=(S_0-D)N(\overline{d_1})-Xe^{-rt}N(\overline{d_2}) \]

е…¶дёӯ

\[ \overline{d_1} = \frac{\ln\left( \frac{S_{0} - D}{X} \right) + \left( r + \frac{\sigma^{2}}{2} \right)t}{\sigma\sqrt{t}} \]

\[ \overline{d_2} = \overline{d_1}- \sigma\sqrt{t} \]

жңүиҝһз»ӯзәўеҲ©зҺҮж”¶зӣҠзҡ„ж¬§ејҸиӮЎзҘЁжңҹжқғзҡ„B-Sе…¬ејҸгҖҒж¬§ејҸиӮЎзҘЁжҢҮж•°жңҹжқғе®ҡд»·е…¬ејҸ

еңЁе®ҡд»·ж Үзҡ„иӮЎзҘЁж”Ҝд»ҳиҝһз»ӯзәўеҲ©зҡ„ж¬§ејҸжңҹжқғж—¶пјҢеҸҜд»ҘжҠҠе®ғеҪ“дҪңж Үзҡ„иӮЎзҘЁдёҚж”Ҝд»ҳзәўеҲ©зҡ„ж¬§ејҸжңҹжқғпјҢеҸӘиҰҒз”Ё\[S_{0}e^{- \text{qT}}\]жӣҝд»ЈеҪ“еүҚиӮЎд»·еҚіеҸҜ

еҚіпјҡ \[ c = S_{0}e^{- \text{qT}}N\left( d_{1} \right) - Xe^{- \text{rT}}N\left( d_{2} \right) \]

е…¶дёӯ

\[ d_{1} = \frac{\ln\left( \frac{S_{0}}{X} \right) + \left( r - q + \frac{\sigma^{2}}{2} \right)T}{\sigma\sqrt{T}} \]

\[ d_{2} = d_{1} - \sigma\sqrt{T} \]

еҠ жӣјпјҚз§‘е°”е“Ҳж №еӨ–жұҮжңҹжқғе®ҡд»·е…¬ејҸ

\[ c = s_{0}e^{- r_{e}T}N\left( d_{1} \right) - Xe^{- r_{c}T}N\left( d_{2} \right) \]

\[ d_{1} = \frac{\ln(\frac{s_{0}e^{- r_{e}T}}{X}) + \{ r_{c} + \frac{\sigma^{2}}{2}\} T}{\sigma\sqrt{T}} \]

\[ d_{2} = d_{1} - \sigma\sqrt{T} \]

е…¶дёӯ\[r_{e}\]иЎЁзӨәеҹәзЎҖиҙ§еёҒзҡ„еҲ©зҺҮпјҢ\[r_{c}\]иЎЁзӨәжҠҘд»·иҙ§еёҒзҡ„еҲ©зҺҮгҖӮ

жңҹиҙ§жңҹжқғвҖ”вҖ”Blackе®ҡд»·жЁЎеһӢпјҲ1976пјү

еҒҮи®ҫжңҹиҙ§д»·ж јиҝҮзЁӢдёә \[ dF = \mu Fdt + \sigma Fdz \]

еҒҮе®ҡжңҹиҙ§еҗҲзәҰе’ҢжңҹжқғеҗҲзәҰеҗҢж—¶еҲ°жңҹпјҢеңЁиҝһз»ӯзәўеҲ©зҡ„жңҹжқғе®ҡд»·е…¬ејҸдёӯпјҢз”Ёжңҹиҙ§д»·ж јд»ЈжӣҝиӮЎзҘЁд»·ж јпјҢе№¶дё”з”Ёж— йЈҺйҷ©еҲ©зҺҮ r жӣҝд»ЈзәўеҲ©зҺҮ q пјҢе°ұеҫ—еҲ°жңҹиҙ§жңҹжқғзҡ„е®ҡд»·е…¬ејҸ

\[ c = e^{- \text{rT}}\left\lbrack F_{0}N\left( d_{1} \right) - XN\left( d_{2} \right) \right\rbrack \]

\[ p = e^{- \text{rT}}\left\lbrack XN\left( - d_{2} \right) - F_{0}N\left( - d_{1} \right) \right\rbrack \]

е…¶дёӯ

\[ d_{1} = \frac{\ln\left( \frac{F_{0}}{X} \right) + \frac{\sigma^{2}T}{2}}{\sigma\sqrt{T}} \]

\[ d_{2} = \frac{\ln\left( \frac{F_{0}}{X} \right) - \frac{\sigma^{2}T}{2}}{\sigma\sqrt{T}} = d_{1} - \sigma\sqrt{T} \]

B-S Formula еёҢи…Ҡеӯ—жҜҚзҡ„иЎЁзӨә

жңҹжқғзҡ„еёҢи…Ҡеӯ—жҜҚвҖңе…„ејҹдјҡвҖқпјҡ

  • Delta еҖј: жңҹжқғеҜ№иө„дә§д»·ж јзҡ„еҸҳеҢ–зҺҮ

  • GammaеҖјпјҡ DeltaеҜ№иө„дә§д»·ж јзҡ„еҸҳеҢ–зҺҮ

  • VegaеҖј пјҡжңҹжқғеҜ№иө„дә§д»·ж јжіўеҠЁзҺҮзҡ„еҸҳеҢ–зҺҮ

  • RhoеҖјпјҡ жңҹжқғеҜ№еҲ©зҺҮзҡ„еҸҳеҢ–зҺҮ

  • ThetaеҖј: жңҹжқғеҜ№ж—¶й—ҙзҡ„еҸҳеҢ–зҺҮ

д»ҘзңӢж¶ЁжңҹжқғдёәдҫӢпјҢж— еҲҶзәўжғ…еҶөдёӢпјҢйҰ–е…ҲжҲ‘д»¬жңү \[ C = C\left( X,S_{0},R,\sigma,T \right) = e^{- rT}E[\max(S_T-X,0)] \]

еңЁB-S Formula дёӢжҲ‘д»¬жңү

\[ C = C\left( X,S_{0},R,\sigma,T \right) = S_{0}N\left( d_{1} \right) - Xe^{- rT}N\left( d_{2} \right) \]

еҲҷеёҢи…Ҡеӯ—жҜҚпјҡ

  • Delta

\[ \Delta = \frac{\partial C}{\partial S_{0}} = N\left( d_{1} \right) \]

  • Gamma

\[ \Gamma = \frac{\partial^{2}C}{\partial S_{0}^{2}} = N^{'}\left( d_{1} \right)\frac{1}{S_{0}\sigma\sqrt{T}} \]

  • Theta

\[ \Theta = \frac{\partial C}{\partial T} = \frac{\sigma}{2\sqrt{T}}S_{0}N^{'}\left( d_{1} \right) + Xre^{- rT}N\left( d_{2} \right) \]

  • Vega

\[ V = \frac{\partial C}{\partial\sigma} = S_{0}\sqrt{T}N^{'}\left( d_{1} \right) \]

  • Rho

\[ \rho = \frac{\partial C}{\partial r} = XTe^{- rT}N\left( d_{2} \right) \]
